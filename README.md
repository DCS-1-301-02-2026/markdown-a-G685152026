# 情報工学科の紹介

## 学科の特徴
本学科では、**コンピュータリテラシー**、*工学概論*、プログラミングなどのIT技術を学ぶことができます。
〜〜難しい講義〜〜実際的なプログラミング力やシステム開発の能力を身につけられます。

## 主な学習内容
### 必修科目
1. プログラミング
1. 基礎ゼミ
1. コンピュータリテラシー


### 選択科目
- 情報工学概論
- プログラミング基礎
